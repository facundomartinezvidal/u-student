# u-student

Claude Code plugin for university and course students. Study, summarize course material, create presentations, write academic documents, and prepare for exams — always grounded in your own course content.

## What it does

- Summarize each class from your course schedule (cronograma)
- Create HTML presentations exportable to PDF
- Write TPs and reports section by section
- Study with flashcards, quizzes, and mock exams
- Build a personalized study plan before a parcial
- Practice oral defense of presentations
- Automatically organize all your course files

Always responds in Spanish and prioritizes your own course material over generic knowledge.

## Installation

Requires [Claude Code](https://claude.ai/code).

```bash
claude plugin install u-student@fmartinezvidal-plugins --scope user
```

> If you haven't added the marketplace yet:
> ```bash
> claude plugin marketplace add facundomartinezvidal/claude-plugins
> claude plugin install u-student@fmartinezvidal-plugins --scope user
> ```

## Getting started

Open Claude Code in your subject folder and run:

```
/u-student:init
```

This creates the folder structure, the course schedule template, and a `CLAUDE.md` file that loads automatically every session so Claude knows which subject you're working on.

## Workspace structure

```
mi-materia/
├── CLAUDE.md                    # Auto-loaded subject context (generated by init)
├── cronograma.md                # Class schedule — maps class numbers to topics
├── programa.md                  # Official course syllabus
└── content/
    ├── primer-parcial/
    │   ├── 01-introduccion.pdf
    │   └── 02-tema-dos.md
    └── segundo-parcial/
        └── 03-tema-tres.pdf
```

Name files as `NN-topic-name.ext` so the plugin can automatically match them to their class in the cronograma.

## Commands

| Command | Description |
|---------|-------------|
| `/u-student:init` | Initialize a subject workspace with folder structure and CLAUDE.md |
| `/u-student:organize` | Organize loose files into the standard content structure |
| `/u-student:summarize` | Summarize course material with cronograma coverage tracking |
| `/u-student:study` | Guided step-by-step session to understand a concept or problem |
| `/u-student:quiz` | Interactive quiz — questions without visible answers, feedback after each |
| `/u-student:flashcards` | Flashcard session with score tracking |
| `/u-student:exam` | Full mock partial exam with detailed grading |
| `/u-student:explain` | Feynman technique — you explain, Claude gives targeted feedback |
| `/u-student:mindmap` | Mermaid mind map of course concepts organized by unit |
| `/u-student:compare` | Comparison table between two or more concepts |
| `/u-student:presentation` | HTML presentation with slide brief and speaker notes |
| `/u-student:defense` | Written defense guide or slide-by-slide oral simulation |
| `/u-student:write` | Write TPs, reports, and essays section by section |
| `/u-student:export` | Export .html, .md, or .docx to PDF |

## Agents

Auto-triggered based on what you write:

| Agent | Triggers on |
|-------|------------|
| `tutor` | "no entiendo", "explicame", "qué es X" |
| `study-planner` | "tengo el parcial en X días", "armame un plan de estudio" |
| `tp-reviewer` | "revisá mi trabajo", "corregí el TP" |
| `writing-coach` | "cómo empiezo", "suena bien esto?", "cómo redacto" |
| `research-assistant` | "dónde busco fuentes", "esta fuente sirve?" |
| `exam-analyst` | "tengo parciales viejos", "qué toman siempre" |
| `progress-tracker` | "cuánto me falta", "cómo voy con la materia" |

## Recommended workflow

```bash
# 1. Initialize your subject
/u-student:init

# 2. Add your materials to content/primer-parcial/ and content/segundo-parcial/
#    Naming: 01-topic-name.pdf

# 3. Organize any loose files
/u-student:organize

# 4. Study
/u-student:summarize
/u-student:quiz --parcial 1
/u-student:exam --parcial 1

# 5. Create deliverables
/u-student:presentation
/u-student:write
/u-student:export presentacion.html
```

## Requirements

- [Claude Code](https://claude.ai/code)
- Git is optional — the plugin detects it automatically and adapts
- For PDF export: Chrome, pandoc, or wkhtmltopdf (optional — the plugin provides instructions if none are found)
