# u-student

Claude Code plugin for university and course students. Helps organize subject materials, study, create presentations, write academic documents, and prepare for exams.

## Features

### Commands

| Command | Description |
|---------|-------------|
| `/u-student:init` | Initialize a subject workspace with folder structure, cronograma, and CLAUDE.md |
| `/u-student:organize` | Organize loose files into the standard content structure |
| `/u-student:summarize` | Summarize course material with plan mode and coverage tracking |
| `/u-student:study` | Guided step-by-step session to understand a concept or problem |
| `/u-student:quiz` | Interactive quiz — questions without answers, feedback after each |
| `/u-student:flashcards` | Flashcard session with score tracking |
| `/u-student:exam` | Full simulated partial exam with grading |
| `/u-student:explain` | Feynman technique — student explains, receives targeted feedback |
| `/u-student:mindmap` | Generate a Mermaid mind map of course concepts |
| `/u-student:compare` | Comparison table between two or more concepts |
| `/u-student:presentation` | Create a structured HTML presentation with slide brief and speaker notes |
| `/u-student:defense` | Oral defense plan or slide-by-slide simulation with feedback |
| `/u-student:write` | Write academic documents section by section (TP, informe, ensayo) |
| `/u-student:export` | Export .html, .md, or .docx to PDF |

### Agents (auto-triggered)

| Agent | Triggers on |
|-------|------------|
| `tutor` | "no entiendo", "explicame", "qué es X" |
| `study-planner` | "tengo el parcial en X días", "armame un plan de estudio" |
| `tp-reviewer` | "revisá mi trabajo", "corregí el TP" |
| `writing-coach` | "cómo empiezo", "suena bien esto?", "cómo redacto" |
| `research-assistant` | "dónde busco fuentes", "esta fuente sirve?" |
| `exam-analyst` | "tengo parciales viejos", "qué toman siempre" |
| `progress-tracker` | "cuánto me falta", "cómo voy con la materia" |

## Recommended workflow

```
# 1. Create subject workspace
/u-student:init

# 2. Add your materials to content/primer-parcial/ and content/segundo-parcial/
#    Naming: NN-nombre-del-tema.ext  (e.g. 01-introduccion.pdf)

# 3. Organize any loose files
/u-student:organize

# 4. Study
/u-student:study transformada-de-laplace
/u-student:quiz --parcial 1
/u-student:exam --parcial 1

# 5. Create deliverables
/u-student:presentation
/u-student:write
/u-student:export presentacion-tema.html
```

## Installation

```bash
claude plugin install u-student
```

Or for local development:
```bash
claude --plugin-dir /path/to/u-student
```

## Subject workspace structure

```
mi-materia/
├── CLAUDE.md              # Auto-loaded context (generated by init)
├── cronograma.md          # Class schedule — maps class numbers to topics
├── programa.md            # Official course syllabus
├── progreso.md            # Study progress (generated by progress-tracker)
└── content/
    ├── primer-parcial/
    │   ├── 01-introduccion.pdf
    │   └── 02-tema-dos.md
    └── segundo-parcial/
        └── 03-tema-tres.pdf
```
